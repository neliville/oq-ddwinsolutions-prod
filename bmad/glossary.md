# Glossaire

| Terme | Définition |
| ----- | ---------- |
| BMAD | Méthode de développement assisté par IA (documentation brownfield, workflows, agents). |
| QSE | Qualité, Sécurité, Environnement (domaine métier du projet). |
| Ishikawa | Diagramme causes-effets (arête de poisson) ; outil d’analyse des causes. |
| 5 Pourquoi | Méthode d’analyse des causes racines par enchaînement de « pourquoi ». |
| QQOQCCP | Méthode des 7 questions (Qui, Quoi, Où, Quand, Comment, Combien, Pourquoi). |
| AMDEC | Analyse des Modes de Défaillance, de leurs Effets et de leur Criticité. |
| Pareto | Diagramme 80/20 pour prioriser les causes. |
| 8D | Méthode de résolution de problèmes en 8 disciplines. |
| Lead | Contact capté (email, nom, source, score) pour la machine à leads. |
| Record | Enregistrement générique d’une création utilisateur (titre, contenu, lien User). |
| Asset Mapper | Composant Symfony pour la gestion des assets (JS, CSS/SCSS) sans Node. |
| sass:build | Commande Symfony (symfonycasts/sass-bundle) pour compiler les SCSS en CSS. |
| Brownfield | Projet existant à documenter et faire évoluer (vs greenfield). |
